# ComboLab

A React Native application built with Expo to save and manage your favorite fighting game combos.

## Features

*   View a list of all your saved combos.
*   Add new combos, specifying the game, character, and notation.
*   Edit and delete existing combos.
*   Filter combos by game or character.

## Getting Started

1. Open command prompt and open the directory where your file is.
2. Next type "npx expo start" in order to get the QR code of your app
3. Once QR code has been shown scan it in you Expo Go app
4. Once finished scanning you can now use your app/
